# Bot Discord

Bot de piadas do Discord.

## Install

`npm install`

## Executando

1. Crie um arquivo `.env` , com sua configuração, você tem exemplo em `.env.example`.

```.env
    CLIENT_TOKEN = "Seu token do discord"
```

## START

+ node index.js

## COMANDOS BOT

+ Quem é você?

+ Conte-me uma piada

### Um bot simples para estudo.
